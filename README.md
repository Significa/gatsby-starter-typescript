- [x] Typescript;

- [x] One config file for site-wide settings;
- [x] Sitemap support;
- [ ] Prettier / Lint stage
- [ ] [`size-plugin`](https://github.com/GoogleChromeLabs/size-plugin) to keep an eye on your bundle sizes
- [ ] Support lambda functions
